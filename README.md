### 1. Reverse a String

Reverses a given string by converting it to an array, reversing the array, and joining it back into a string.

### 2. Count Vowels in a String

Counts how many vowels (a, e, i, o, u) exist in a given string using a loop and conditional check.

### 3. Check for Palindrome

Checks whether a string reads the same forward and backward by comparing it with its reversed version.

### 4. Find the Maximum Number

Finds the largest number in an array using JavaScript built-in functionality.

### 5. Remove Duplicates from an Array

Removes duplicate values from an array using `Set`, which stores only unique values.

### 6. Sum of All Numbers in an Array

Calculates the total sum of all numbers in an array using a loop.
